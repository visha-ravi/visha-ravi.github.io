About

Console based application where user maintains the records of the football clubs who are participating in the Premier League Championship. Using this console application user can add the details of the football club, delete the details of the football clubs, able to see the statistics of each football club by entering their club name, able to see the premier league table, add played match between two clubs and user can view the premier league table in GUI. In the gui user can generate random match between two clubs, user can sort score according to some criterias and user can save the club details.
